> Paysera Library for Node.js

## Usage

Install package:

```sh
# npm
npm install nodesera

# yarn
yarn add nodesera

# pnpm
pnpm install nodesera
```

Import:

```js
// ESM
import { } from 'nodesera'

// CommonJS
const { } = require('nodesera')
```
